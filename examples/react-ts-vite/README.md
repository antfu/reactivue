This is a demo app for [Reactivue](https://github.com/antfu/reactivue) library. Reactivue makes Vue Compostion API and Vue plugins work in React apps.

## How to use

```bash
npx degit antfu/reactivue/examples/react-ts-vite
cd react-ts-vite
yarn install
yarn dev
```
